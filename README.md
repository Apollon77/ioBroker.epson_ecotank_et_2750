## Seq adapter for ioBroker

This adapter read tank level and other information from EPSON EcoTank ET-2750 and store in ioBroker.

## Configuration

1. Create a new instance of the adapter
2. Fill the URL/IP and port of the EPSON EcoTank ET-2750
3. Configur ther Synctime (default 180 minutes)
4. Save the settings
